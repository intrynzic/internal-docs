# System Requirements

This document outlines the minimum and recommended system specifications required to build and run Intricate Engine.

---

## Supported Operating Systems

Intricate only runs on **Windows 10 and 11**.

---

## Hardware Requirements

| Component      | Minimum                          | Recommended                      |
|----------------|----------------------------------|----------------------------------|
| CPU            | Quad-core 2.0 GHz                | 6+ cores, 3.0+ GHz, modern gen   |
| RAM            | 8 GB                             | 16+ GB                           |
| GPU            | Any GPU with Vulkan 1.3 support  | Discrete GPU with 4+ GB VRAM     |
| Storage        | 20 GB free disk space            | -                                |
| Display        | 1080p                            | 1440p or higher                  |

---

## Software Requirements

- **Git 2.5** (Windows) or newer
- **GitHub Desktop**
- **Python 3.12** or newer
- **Vulkan SDK 1.3.216**
- **DirectX 11 & 12**

!!! info "IDE requirements"
    See [IDE requirements](ide-requirements.md) for more details.
